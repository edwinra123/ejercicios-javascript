# Ejercicios de JavaScript

Repositorio con ejercicios prácticos de JavaScript para aprender y reforzar conceptos de programación web.

---

## Contenido

Este repositorio incluye ejercicios sobre:

- Variables y tipos de datos
- Operadores
- Condicionales (`if`, `else`, `switch`)
- Bucles (`for`, `while`, `do...while`)
- Funciones
- Arrays 
- Objetos
